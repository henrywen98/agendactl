# ADR-0004: 删独立 probe，拆为薄容器列举命令

- 状态：accepted
- 日期：2026-06-25
- 上下文：CLI-only（[ADR-0002]）下，agent 对 `probe` 的真实刚需只剩**拿到准确容器名**（写 `--calendar "工作"` 别写成 `"Work"`）。计数 / 今日数它用 `list --due today` 自己就能数。v1.0 的 `probe`（三 App 概览 + 计数 + 今日数 + 5min 缓存，§5.3）属超配。
- 决策：删除独立 `probe` 组件。改为 `apple reminders lists` / `apple calendar calendars` / `apple notes folders`，**只回真实容器名**（+ 可选计数）。**探针优先原则保留**（写前先拿准名字），但载体从专门 probe 降为薄列举命令。
- 后果：删 §5.3、§7.2 的 probe 组件、O5 缓存规格。`lib/std.js` 进一步瘦身（`safeWhose` / 概览聚合都不需要）。
